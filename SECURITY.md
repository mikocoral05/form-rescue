# Security Policy

## Supported Versions

We currently support security updates for the following versions of `form-rescue`:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within `form-rescue`, please **do not open a public issue**.
Instead, please report it via GitHub's private vulnerability reporting feature located under the "Security" tab of this repository, or reach out to the maintainer directly. We will address the issue as quickly as possible.
