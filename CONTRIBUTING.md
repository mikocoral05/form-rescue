# Contributing to form-rescue

First off, thank you for considering contributing to `form-rescue`! It's people like you that make open-source software such a great community.

## Local Development Setup

1. Fork and clone the repository.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development compiler in watch mode:
   ```bash
   npm run dev
   ```
4. Open the `examples/index.html` file in your browser (using a tool like `live-server`) to test your changes interactively.
5. Run the Jest test suite to ensure everything is passing before submitting:
   ```bash
   npm test
   ```

## Pull Request Process

1. Ensure your commits are descriptive.
2. Make sure all existing and new tests pass successfully.
3. Open a Pull Request using the provided template. If it's a visual change or adds a complex feature, providing a code snippet in the PR description is highly appreciated!
